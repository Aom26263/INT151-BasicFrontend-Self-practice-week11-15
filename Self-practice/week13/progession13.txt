- เริ่มจากการทำปุ่ม load-btn ให้กดเเล้วโหลดข้อมูลลงมา โดยการ fetch ข้อมูลจาก backend มาเเล้วสร้างตัวเเปรมาเก็บค่าข้อมูลที่ fetch มา (student) จากนั้นก็สร้างตัวเเปรที่เรียก table 
ใน html มา เเละเอาข้อมูลที่ fetch มาเอาลงมาใส่ในตารางที่ละตัวด้วยการใช้ forEach เเละสร้าง element html ก่อนคือ tr เเละค่อยสร้าง tr.innerHTML ข้างในของมัน
ก็สร้าง td เเล้วเอาข้อมูลที่ได้มา มาใส่เเต่ละช่อง เเละสร้างปุ่ม edit ไว้ด้วย table.appendChild(tr);เพิ่มเข้าไปใน tr จากนั้นก็สร้างปุ่ม edit ให้กดเเล้วจะสามารถเเก้ไขข้อมูลได้
เเละมีปุ่ม save กับ cancel มาเเทน โดยเรียกผ่าน function enterEditMode ใน function จะเอาข้อมูลเเต่ละตัวมาปรับให้่เป็น value ก่อน เเละเปลี่ยนปุ่ม edit ให้เป็น
ปุ่ม save กับ cancel เเทน โดยการ const row = btn.closest("tr"); กำหนดตัวเเปรที่เรียก tr ที่ใกล้ที่สุดก่อน เเละ row.children[1].innerHTML = `<input class="input-edit" value="${oldName}">`
ค่อยปรับตามเเต่ละ row โดยใช้ children[1] บอกว่าตัวไหน จากนั้นก็สร้างปุ่ม save กับ cancel ปุ่ม save เมื่อกดจะเรียก function saveEdit จะรับพารามิเตอร์มาเป็น row กับ id
เเละ fetch ไปหา id เเต่ละจุดโดยใช้ method put เพื่อทึี่จะเเก้ไขข้อมูลได้ เเละเมื่อเเก้ไขเสร็จกด save ก็จะเปลี่ยนกลับเป็นปุ่ม edit เหมือนเดิม ส่วนปุ่ม cancel ก็จะเขียน 
function คล้าย save เเต่เเค่ไม่ได้มีการเเก้ไขข้อมูล เมื่อกดเเล้วจะเปี่ยนปุ่มกลับไปเป็น edit เหมือนเดิม
